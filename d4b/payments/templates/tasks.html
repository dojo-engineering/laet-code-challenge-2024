The challenge: finish this website!

  1. Have a look around the website and see what works and what doesn't
  2. Go to the helpers.py file and start implementing the functions
  3. Have fun!

Hints:

  1. You don't need to run your code each time you change it, just refresh the website (press command-R)
  2. Read the comments for each function before you start implementing it
  3. Ask Joel, Daniel Gunjan, Emma and Savannah lots of questions
  4. Use Google if you can't remember the right syntax for something (professional software developers use Google constantly)
  5. Name your variables well and your code will be easier to debug
  6. Simple code is good code (and if you try something fancy it might be hard to debug)
