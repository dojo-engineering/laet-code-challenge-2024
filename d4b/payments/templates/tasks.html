The challenge: finish this website!

  1. Have a look around the website and see what works and what doesn't
  2. Go to the helpers.py file and start implementing the functions
  3. If you implement all of the functions, talk to one of your facilitators and we'll set you a new challenge
  4. Have fun!

Hints:

  1. You don't need to run your code each time you change it, just refresh the website (press command-R)
  2. Don't forget to save your code before you refresh the website (command-S)
  3. If you print() a variable it will come out in the terminal in VS Code
  4. Read the comments for each function before you start implementing it
  5. Ask your facilitators lots of questions
  6. Use Google if you can't remember the right syntax for something (professional software developers use Google constantly)
  7. Name your variables well and your code will be easier to debug
  8. Simple code is good code (and if you try something fancy it might be hard to debug)
